# 使用next.js实现React的SSR技术

## 技术栈：
next.js + react

## 下载

 	git clone https://github.com/Topthinking/react-next-app.git

 	cd react-next-app

 	npm install

 ## 运行（nodejs 8.0+）
```
 npm run dev (正常编译模式)

 访问 http://localhost:3000

```
